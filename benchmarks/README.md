# Benchmark Contracts

This directory stores reviewable promotion benchmark contracts.

Accepted promotion benchmarks must use candidate-policy replay over the
canonical fixed window `2021-01-01` through `2026-01-01` end-exclusive. This
covers the full 2021-2025 calendar years and 1255 expected NYSE trading days.
Contracts must not preselect final target tickers or carry fixed target/window
component panels. Candidate contracts must pass the benchmark validator before
freeze.
